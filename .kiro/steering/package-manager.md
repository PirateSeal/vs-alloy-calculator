# Package Manager

This project uses **pnpm** as the package manager.

## Commands

Always use pnpm for package management operations:

- Install dependencies: `pnpm install`
- Add packages: `pnpm add <package>`
- Add dev dependencies: `pnpm add -D <package>`
- Remove packages: `pnpm remove <package>`
- Run scripts: `pnpm run <script>` or `pnpm <script>`
- Run tests: `pnpm test`
- Build: `pnpm build`

## Important

Never use `npm` or `yarn` commands. Always use `pnpm` instead.
